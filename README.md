# Анализ зарплат в сфере Data Science

### О проекте

В рамках проекта проведён анализ датасета с зарплатами специалистов в области Data Science с целью выявления факторов, влияющих на уровень дохода.

### Задачи

* Определить наиболее высокооплачиваемые должности
* Проанализировать влияние удалённой работы
* Оценить влияние страны и размера компании
* Сравнить динамику зарплат (2020–2023)
* Сформировать портрет наиболее успешного специалиста
* Проанализировать международную занятость

### Используемые технологии

* Python (pandas, numpy)
* matplotlib, seaborn
* Jupyter Notebook

### Основные результаты

* Высокие зарплаты сконцентрированы в США (~91% среди топ-5%)
* Рынок смещается в сторону AI/ML ролей
* Удалённая работа не оказывает значимого влияния на зарплату
* Международная занятость встречается редко (~3–4%) и реализуется преимущественно через удалёнку (~74%)
* Высокий доход связан с технической экспертизой и уровнем Senior

### Портрет успешного специалиста

Типичный высокооплачиваемый специалист — это Senior Data/ML специалист (например, Data Engineer, Data Scientist или Machine Learning Engineer), работающий в компании из США. Формат работы может быть как офисным, так и удалённым.

# Data Science Salaries Analysis

### Project Overview

This project analyzes salary data for Data Science professionals to identify key factors that influence compensation, including role, experience level, location, and work format.

### Objectives

* Identify the highest-paying roles
* Analyze the impact of remote work on salaries
* Evaluate the influence of company size and location
* Compare salary dynamics between 2020 and 2023
* Define a profile of the highest-paid specialists
* Explore international employment patterns

### Tools and Technologies

* Python (pandas, numpy)
* matplotlib, seaborn
* Jupyter Notebook

### Key Findings

* High salaries are heavily concentrated in the United States (~91% among top 5%)
* The market shows a strong shift toward AI/ML roles
* Remote work itself does not significantly affect salary levels
* Cross-country employment is rare (~3–4%) and mostly remote (~74%)
* High earnings are strongly associated with technical expertise and senior-level experience

### High-Earning Specialist Profile

A typical high-earning Data Science professional is a Senior-level specialist in a technical role (e.g., Data Engineer, Data Scientist, Machine Learning Engineer), working for a US-based company. Work format can be either remote or on-site.
